# Cleanup

Clean up build artifacts, old data, and system resources.

## When to use
When disk space is low or periodically. Invoke with `/cleanup`.

## Instructions
<!-- Customize for your project -->
1. **Build artifacts** — remove old builds, caches
2. **Old data** — prune logs, old backups
3. **Dependencies** — remove unused packages
4. **Report** — show space before and after
